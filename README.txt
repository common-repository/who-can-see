Manipulate part of page/post to be showed/hidden for some roles

== Description ==
This plugin allows you to manage the visibility of your page/post the way that will be showed to specified with :
[whocansee only='visitor|administrator']something to be showed to visitors/administrators alone [/whocansee]
OR you can use the except to exclude who cant see the content.
[whocansee except='visitor']something to be showed to  everyone except visitors[/whocansee]
roles :

* visitor
* subscriber
* contributor
* author
* editor
* administrator

== Installation ==
1. Upload "who-can-see" folder to the "wp-content/plugins" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
